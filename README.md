# LoL-Monitor (英雄联盟战绩微信推送助手)
#(必须先登陆客户端才能monitor!)

一个轻量级的 Python 后台脚本。通过静默读取本地英雄联盟客户端（LCU）接口，全自动监控好友的游戏状态，并在游戏结束时将 **KDA、胜负、游戏时长** 等战报实时推送到你的微信。

全程无需申请 Riot 官方 API，零内存占用，支持开机自启与无感后台运行。

## ✨ 核心特性

* **🚀 寄生式本地读取**：无需输入账号密码，直接挂载本地 `LeagueClientUx.exe` 端口，安全防封。
* **📱 微信实时推送**：接入 Server酱，游戏结束瞬间手机收到战绩播报。
* **🎯 专属白名单**：支持设置监控列表，只盯你在乎的玩伴，拒绝被几百个好友刷屏。
* **👻 无感开机自启**：一键写入 Windows 注册表，开机自动潜伏，打开游戏自动苏醒。
* **📊 本地日志记录**：自动生成 `lol_log.csv` 文件，记录好友上下线与游戏历史。
